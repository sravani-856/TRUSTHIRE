# TRUSTHIRE

## Smart Web-Based Recruitment with Resume Management

TRUSTHIRE is a web-based recruitment platform that connects Job Seekers and Recruiters. It includes intelligent resume analysis to improve job matching and make recruitment more transparent.

## Features

- User Registration & Login (OTP Verification)
- Resume Upload
- ATS Resume Score
- Job Match Score
- Skill Gap Analyzer
- Explainable Match
- Job Recommendations
- Recruiter Dashboard
- Admin Dashboard
- Job Posting & Approval
- Application Tracking

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- JWT Authentication
- Multer
- PDF Parser

### Database
- MySQL

## Installation

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm start
```
