# 🚀 TRUSTHIRE

## Smart Web-Based Recruitment with Resume Management

TRUSTHIRE is a full-stack web-based recruitment platform that connects **Job Seekers**, **Recruiters**, and **Administrators**.

The platform intelligently analyzes resumes and recommends the most suitable jobs based on candidate skills, making recruitment smarter and more transparent.

---

## ✨ Features

### 👨‍🎓 Job Seeker
- User Registration & Login
- Email OTP Verification
- Resume Upload
- ATS Resume Score
- Job Match Score
- Skill Gap Analyzer
- Explainable Match
- Recommended Jobs
- Apply for Jobs
- Application Tracking

### 🏢 Recruiter
- Recruiter Dashboard
- Post Jobs
- View Applicants
- Manage Job Listings

### 🛡️ Admin
- Admin Dashboard
- Approve Recruiters
- Manage Jobs
- Monitor Platform
